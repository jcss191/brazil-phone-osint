# 🛡️ Brazil Phone OSINT - Deep Search Engine

### ⚡ A Ferramenta
Desenvolvida para analistas de **Cyber Intelligence**, esta ferramenta automatiza o processo de **Deep Web Scraping** e **Google Dorks** focado em telefones brasileiros.

### 🔍 Funcionalidades de Elite
- **Dorking Automático:** Localiza o número em arquivos `PDF`, `XLSX` e `DOCX` expostos na rede.
- **Identificação de Alvo:** Separação automática de DDD e região.
- **Login Seguro:** Interface de terminal com controle de acesso.
- **Automação de Browser:** Abre múltiplas camadas de busca simultaneamente.
